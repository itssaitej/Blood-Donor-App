# Blood Donor App 🩸

A Spring Boot backend application for managing blood donors.

## Features
- User Registration & Login
- Donor CRUD Operations
- Search by City, Area, Blood Group
- Availability Tracking

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## API Examples

### Register
POST /auth/register

### Get Donors
GET /donors

## Future Improvements
